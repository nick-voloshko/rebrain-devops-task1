# SCalc App

Отличный простой калькулятор.
> These guys see the forest for the trees. -- `L.T.`  

## Что умеет: 
1. Складывать
2. Вычитать
3. Умножать (скоро)
4. Делить (скоро)

### Установка
``$ apt intall scalc``

## Использование
* Сложить два числа ``$ scalc one plus two``
* Вычесть из одного числа другое ``$ scalc two minus one``
